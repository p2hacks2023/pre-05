# P2HACKS2023 アピールシート 

プロダクト名  
ねこサマー 

コンセプト  
かわいい猫と涼しい空間で過ごして癒されるこで涼しさを感じると共に心もクールダウンできる

対象ユーザ  
・涼しさを感じたい人
・猫が好きな人
・ストレスが溜まっている/癒しが必要な人

利用の流れ  
本体(exeファイル)を起動することで利用可能。
1プレイ当たりの時間が短いので、気軽にできる

推しポイント
